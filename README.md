# Sumo_TechDesign_Test

Technical Test
Part One
In Unreal Engine 5 create a project using the third person template:
1. Add a basic health system to the third person character
2. Create a gameplay object that the player must overcome or avoid that can
damage the player’s health.
In the Blueprint, document any design choices you have made, and why you have made
them.
Part Two
Take the gameplay object you have created in part one and create a version or modify the
gameplay object so that it works in a multiplayer environment.
